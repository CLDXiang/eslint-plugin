# 在 webstorm 中使用 eslint-plugin-lint-md

## 设置 .md 后缀支持

- 进入 `file -> settings -> languages & frameworks -> JavaScript -> Code Quality Tools -> ESLint` 设置项
- 在 `Run for files` 一栏中添加 `.md` 后缀。
- 点击 ok 保存设置
